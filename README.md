## About This Repository

This repository is a structured, end-to-end learning path for **Cloud Computing concepts** and **AWS (Amazon Web Services)** — designed to take anyone from zero knowledge to mastery level, one concept at a time.

It is built around two parallel tracks:

1. **Cloud Computing Lessons and Fundamentals** — the *theory*: what cloud computing is, how it works, its paradigms (edge, fog, mobile, etc.), architecture patterns, security, economics, and emerging trends.
2. **AWS Guide Step by Step** — the *practice*: a hands-on, service-by-service walkthrough of AWS, from account setup to certification-level mastery, with lessons paired with practical activities.

Each lesson is self-contained, descriptively named, and ordered so that no concept depends on something not yet covered.

## How to Use This Repository

1. **Start with Cloud Computing Fundamentals** — even if you only care about AWS, the theory here gives you the *why* behind every AWS service.
2. **Move to the AWS Step-by-Step Guide** — follow folders in numerical order. Each main folder contains topic-specific lesson subfolders.
3. **Each lesson subfolder contains:**
   - `lesson.md` — concept explanation, key terms, and exam/industry-relevant points
   - `activity.md` — hands-on practice, console/CLI exercises, or review questions to reinforce the lesson
4. **Track your progress** — each folder includes a `README.md` with a checklist of lessons. Mark items as you complete them.
5. **Revisit and reinforce** — concepts build on each other; revisiting earlier folders while progressing through later ones is encouraged.

## Track 1: Cloud Computing Lessons and Fundamentals

A 40-folder theory track covering:

## Learning Philosophy

- **Step by step** — no concept appears before its prerequisites.
- **Theory and Practice** — every AWS lesson pairs with a hands-on activity.
- **Descriptive over clever** — folder and file names describe exactly what's inside.
- **Mastery-oriented** — covers beginner basics through certification-level and real-world architecture depth.
- **Self-contained** — each lesson can be read independently for quick reference, while still fitting into the larger learning path.
